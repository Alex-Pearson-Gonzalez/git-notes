# Git Basics

## git init
Turns a folder into a Git repo.
вЂ‹```bash
git init
вЂ‹```

## git clone
Downloads a copy of a remote repo (like from GitHub) to your computer.
вЂ‹```bash
git clone https://github.com/user/repo.git
вЂ‹```

## git status
Shows what's changed вЂ” modified, staged, or untracked files.
вЂ‹```bash
git status
вЂ‹```

## git add
Stages changes, marking them ready to be committed.
вЂ‹```bash
git add filename.md
git add .          # stage everything
вЂ‹```

## git commit
Saves a snapshot of staged changes with a message.
вЂ‹```bash
git commit -m "Describe what changed"
вЂ‹```

## git push
Uploads your local commits to GitHub.
вЂ‹```bash
git push
вЂ‹```

## git pull
Downloads and merges the latest changes from GitHub into your local copy.
вЂ‹```bash
git pull
вЂ‹```
